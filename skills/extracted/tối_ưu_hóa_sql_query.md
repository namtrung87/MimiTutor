## Tối ưu hóa SQL Query
### Mô tả
Tối ưu hóa SQL query để tăng tốc độ thực hiện bằng cách thêm index vào cột và sử dụng lệnh EXPLAIN ANALYZE để phân tích hiệu suất.

### Các bước thực hiện
1. Xác định cột cần được thêm index.
2. Thêm index vào cột đã xác định.
3. Sử dụng lệnh EXPLAIN ANALYZE để phân tích hiệu suất của query.

### Ví dụ
Khi cần tối ưu hóa một SQL query, bạn có thể thêm index vào cột 'user_id' và sau đó sử dụng lệnh EXPLAIN ANALYZE để phân tích hiệu suất của query, như đã đề cập trong câu trả lời trước: "Bản nên thêm index vào cột 'user_id' và sử dụng EXPLAIN ANALYZE."
